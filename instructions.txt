Examination Instructions
You are provided with a fully working National E-Voting console application packed into one file. It
handles candidate CRUD (with eligibility checks for age, education, criminal record), voting station
management, position and poll lifecycle, voter registration and verification, ballot casting with du-
plicate prevention, result tallying with bar charts and turnout statistics, and role-based access for
four admin types plus voters.
Run it first. Log in as admin, create a station, register a voter, set up a poll end-to-end, then vote.
Understand every feature before you touch the code.
The application was deliberately written to violate every principle you have studied and Your Task
is to refactor this monolith into a modular, object-oriented Python project. The application must
behave identically after refactoring e.g., same menus, same prompts, same outputs. Do not add new
features.
Apply the following principles from this course:
1.Modular Design
2.Object-Oriented Design
3.Separation of Concerns
4.Clean Code
